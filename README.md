# Application-tracker
The Job Application Tracker is a simple and efficient web application designed to help users manage and track their job applications in one place. It allows users to record application details and monitor the status of each application throughout the job search process.

Add job applications with company name, role, date, and status

View all job applications in a list format

Filter applications by status (Applied, Interview, Selected, Rejected)

Delete job applications

Persistent data storage using browser LocalStorage
